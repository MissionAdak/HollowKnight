
# Hollow Knight Game

A 2D platformer game inspired by Hollow Knight, featuring a menu system and game mechanics.

## Features

- Interactive menu system with music
- Player movement and combat mechanics
- Multiple levels with different challenges
- Audio effects for various game actions

## Controls

- Arrow keys: Move the character
- Space: Jump
- Z: Attack
- ESC: Pause/Menu
